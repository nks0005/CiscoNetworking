@ Internet
	인터(Inter)라는 의미는 '연결'을 의미합니다.
		International 이란 '국제적인', '국제간의'라는 뜻으로 해석됩니다.
		즉, Internatona이란 여러 나라를 묶을 때 사용하는 말입니다.
	# 인터넷(Internet)이란, '여러 개의 네트워크를 묶었다.'는 의미를 가지고 있습니다.

@ Internet 특징
	1. 하나의 프로토콜만을 사용합니다. ( TCP/IP )
	2. 웹브라우저를 통해 인터넷을 탐험.
	3. 인터넷에는 없는 정보가 거의없음.

@ Web
	www -> World Wide Web

@ IntraNet (인트라넷)
	Intra - 내부의
	즉, 내부의 네트워크(사내 네트워크)

@ ExtraNet (엑스트라넷)
	협력 회사나 고객에게 사용할 수 있도록 한 것.

 인터넷은 네트워크를 여러 개 묶어놓은 네트워크 연합을 말하고, 이것들은 TCP/IP란 공통의 프로토콜을 사용합니다.
	인트라넷은 회사에서 쓰는 여러 가지 프로그램들을 마치 인터넷을 사용하는 것처럼 쓰드록 만들어놓은 것인데,
	인트라넷은 그 회사의 직원 이외에는 사용할 수가 없다.

	엑스트라넷은 그 사용 범위를 직원 이외에도 협력 회사나 고객까지로 확대한 개념입니다.
